
/**
 * Installation
*/

* install module by running following command:
   git clone https://github.com/poojasharma48/location_time.git
   
* Enable Location and Time module as usual.

* After installation , On config page: https://your_domain_name/admin/config
   you will see "Configure Location settings" under "Content authoring" section.

* After click on "Configure Location settings" link , you will navigate to this page:
   https://your_domain_name/admin/config/custom/regional/settings

   On this page following field display: Country , City, timezone.

   This is the page from where you can add country, city , timezone to display on the site.

* To display the Country, City and time , place the "Location and time" block as usual.

* You can check output in provided screnshot as well
